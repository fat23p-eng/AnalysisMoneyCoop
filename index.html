ข้อมูลจะถูกส่งเข้า Sheets ทุกครั้งที่กด Sync
วิธีตั้งค่า Google Sheets
1
เปิด Google Sheets และสร้าง Spreadsheet ใหม่ หรือใช้ที่มีอยู่แล้ว
2
ไปที่ Extensions → Apps Script แล้ววาง Script ที่ copy ด้านบน
3
กด Deploy → New deployment เลือก type เป็น Web app และ access เป็น Anyone
4
Copy Deployment URL มาวางในช่อง Script URL ด้านบน
5
กด Sync ข้อมูลทั้งหมด ระบบจะส่งข้อมูลงบการเงินทั้งหมดเข้า Google Sheets
